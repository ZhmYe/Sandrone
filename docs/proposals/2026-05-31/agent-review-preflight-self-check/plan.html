Agent Review Preflight Self Check

让 agent 在提交给 reviewer 前先按对应 reviewer 标准自检，减少必然失败的 review 调用。

流程

  1. 测试锁定默认 prompt 必须包含自检要求。
  2. issue-agent.md 说明共享契约职责。
  3. plan-agent.md 增加 PlanReviewer preflight。
  4. implementation-agent.md 增加 TestReviewer 和 DesignReviewer preflight。
  5. README 与 Skill 同步说明。
