Failure Path Error Assertions

本计划把失败路径测试收紧为明确错误匹配，特别覆盖 review gate 和 approval gate。

目标

  • 所有 intentional failure tests 必须匹配 stderr 中的预期错误。
  • review gate 失败必须匹配具体 reviewer，例如 PlanReviewer rejected。
  • approval gate 失败必须匹配缺失或过期 approval 的错误文本。

验证

  • cargo fmt --check
  • cargo check
  • cargo clippy --all-targets -- -D warnings
  • cargo test
  • python3 scripts/validate_proposals.py
